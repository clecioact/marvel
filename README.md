## Clone este repositório
$ git clone <https://github.com/clecioact/marvel>

## Acesse a pasta do projeto no terminal/cmd
$ cd marvel

## Instale as dependências
$ npm install

## Para rodar o server
$ node server

## O servidor inciará na porta:8080 - acesse <http://localhost:8080>
